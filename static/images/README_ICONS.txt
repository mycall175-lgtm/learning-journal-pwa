PWA Icons Required
==================

For full PWA functionality, add the following icon files to this directory:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these icons using:
1. Online tools like https://realfavicongenerator.net/
2. PWA Asset Generator: https://github.com/onderceylan/pwa-asset-generator
3. Any image editor (create a square image and resize)

The app will work without icons, but PWA installation may not be fully functional.
