
# Learning Journal PWA (skeleton)

Files included:
- index.html, journal.html, projects.html, about.html
- css/style.css (mobile-first responsive)
- js/main.js (menu toggle)
- manifest.json (basic PWA manifest)

How to use:
- Unzip and open the folder in VS Code: `code learning-journal-pwa`
- Serve locally (recommended) to test PWA features. Simple options:
  - Using Python 3: `python -m http.server 8000` then open http://localhost:8000
  - Using VS Code Live Server extension
- Commit the folder to GitHub and enable GitHub Pages, or deploy to Netlify/Vercel.
